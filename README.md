# Stock Predictor

Machine learning model that predicts if a stock price will go up or down.

# How to run:
1. Install requirements: pip install pandas numpy yfinance xgboost scikit-learn matplotlib
2. Run: python Stock_Predictor.py
3. Enter a stock symbol

# What it does:
- Downloads stock data
- Calculates technical indicators (RSI, MACD, moving averages)
- Trains XGBoost model
- Shows prediction with confidence level
- Creates charts showing results

Built with Python and XGBoost.
